# Assignment 1 - Game Programming - Đập đầu Zombies
Để chạy game này, bạn cần cài đặt Pygame trên máy tính của mình. Hãy làm theo các bước sau:
+ Đảm bảo rằng bạn đã cài đặt Python trên máy tính của mình. 

+ Mở Terminal hoặc Command Prompt trên máy tính của bạn.

+ Sử dụng pip, công cụ quản lý gói Python, để cài đặt Pygame. Gõ lệnh sau đây và nhấn Enter:

        pip install pygame
